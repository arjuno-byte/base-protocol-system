# Bug Bounty And Responsible Disclosure

BPS welcomes responsible security research.

## In Scope

- Consensus safety issues.
- Transaction validation bugs.
- Mint hard-cap bypasses.
- Unauthorized fund movement.
- Remote crash or denial-of-service vectors in public node paths.
- Release artifact integrity issues.

## Out Of Scope

- Social engineering.
- Spam against public endpoints.
- Issues requiring leaked private keys or mnemonics.
- Automated high-volume testing against public infrastructure without permission.

## Reporting

Use the private process in `SECURITY.md`. Include:

- Impact summary.
- Reproduction steps.
- Affected commit or release.
- Suggested mitigation if known.

## Rewards

No guaranteed paid bounty pool is published yet. Valid reports are still welcome
and will be credited if the reporter wants public acknowledgement after the fix.
