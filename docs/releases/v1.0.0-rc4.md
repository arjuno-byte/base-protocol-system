# v1.0.0-rc4 Release Notes

Release date: 2026-04-29

## Highlights

- Adds public BPS-01 endpoint metadata.
- Publishes snapshot documentation and checksum.
- Documents faucet, explorer, and status page endpoints.
- Adds governance, validator onboarding, slashing, RPC, monitoring, disaster
  recovery, audit status, bug bounty, and chain registry documentation.
- Updates release automation to generate SHA-256 checksums and Sigstore bundles.

## Known Limitations

- Public raw P2P peers must be announced as `<node_id>@IP_PUBLIC:26656`.
- Public RPC, faucet, explorer, status page, and snapshots are optional
  app-layer services and are not required for core node sync.
